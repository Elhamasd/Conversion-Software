Headers

You can create a hierarchy of headers using # symbols: # Header 1 ## Header 2 ### Header 3 #### Header 4 ##### Header 5 ###### Header 6

Emphasis

  • Bold text is created with ** or __.
  • Italic text is created with * or _.
  • Combined bold and italic can be done by nesting.
